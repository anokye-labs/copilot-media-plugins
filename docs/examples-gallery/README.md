## How to Use These Examples

1. **Read the goal** — each example starts with what it produces
2. **Copy the commands** — examples use real parameters you can paste directly
3. **Modify for your needs** — swap prompts, paths, and parameters
4. **Chain examples** — combine techniques from different examples into your own workflows
